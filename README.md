# MusicApp

#### 介绍

使用开源`网易云音乐 API`接口 [NeteaseCloudMusicApi]( https://github.com/Binaryify/NeteaseCloudMusicApi) 

本人wpf 新手,也是边写边学习中,感觉写的很乱，功能虽已实现,但是会不断优化代码
